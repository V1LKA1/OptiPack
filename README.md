# ОптиПак
OptiPack—сборка для Minecraft, созданная для стабильной игры и мультимального FPS. Никаких лишних модов — только правенные решении: натрий, литий, немедленнобыстро и быстро. Подходит для Ткань 1.21.1 Лёгкая установка, чистая структура, безопасный запуск.

#Уствновка
1. Установи Fabric
2. Скачай архив
3. Распакуй в папку "mods"
4. Запусти игру
